# PENTeam Agent Skills Index
# Reusable skill files for mathematical research tasks

| Skill | File | Description |
|-------|------|-------------|
| Mathematical Theorem | `math-theorem.md` | Guidelines for formulating theorems and proofs |
| Code Implementation | `math-code.md` | Best practices for translating math to Python |
| Testing | `math-testing.md` | Comprehensive testing strategies |
| Review | `math-review.md` | Critical review guidelines |
| Coordination | `coordination.md` | Multi-agent orchestration and project management |

## Usage

Agents can reference these skills for guidance on specific tasks.
The Supervisor will ensure appropriate skills are available for each phase.

## Adding New Skills

Create new skills in this directory following the naming convention:
`[skill-category]-[skill-name].md`

Then add an entry to this index file.
